Exercice 2 - Créer un formulaire flexible

Bloc de travail

Créer une page HTML en ajoutant le code suivant :

  <div class="container">
  <form  action="mailto:adresse@email.tld" method="post">
    <ul class="flex-form">
      <li>
        <label for="nom">Nom</label>
        <input type="text" id="nom" placeholder="Entrez votre nom ici">
      </li>
      <li>
        <label for="prenom">Prénom</label>
        <input type="text" id="prenom" placeholder="Entrez votre prénom ici">
      </li>
      <li>
        <label for="email">Email</label>
        <input type="email" id="email" placeholder="Entrez votre email ici">
      </li>
      <li>
        <label for="telephone">Téléphone</label>
        <input type="tel" id="telephone" placeholder="Entrez votre téléphone ici">
      </li>
      <li>
        <label for="message">Message</label>
        <textarea rows="6" id="message" placeholder="Entrez votre message ici"></textarea>
      </li>     
      <li>
        <button type="submit">Envoyer</button>
      </li>
    </ul>
  </form>
</div>
  1. Quel est le rôle de la balise <form> ?
  2. Que fait la propriété action ?
  3. Quel est le rôle de l'instruction mailto ?
  4. Quel est la différence entre la balise <label> et la balise <input> ?

Utilisez une nouvelle page css et ajouter le code suivant :

	/************* Gestion de la présentation (layout) *************/
.flex-form li{
	display: flex;
	flex-wrap: wrap; /* */	
	justify-content: flex-start;/*  */
	align-items: center;/*  */
}
	/* la taille du label va de 120 à 220px */
.flex-form > li > label {
	flex-grow: 1;/*  */
	flex-shrink: 0;/*  */
	flex-basis:120px;/*  */	 
	max-width: 220px;/*  */	
}
/* la taille de la saisie est de 220px la taille totale de la ligne est de 340px*/
.flex-form > li > label + *{
	flex-grow: 1;/*  */
	flex-shrink: 0;/* */
	flex-basis:220px;/*  */	
}
	/************* Gestion des styles *************/
body {
  background: #f2f2f2;
}	
.flex-form li button {
  margin-left: auto;
  padding: 8px 16px;
  border: none;
  background: #333;
  color: #f2f2f2;
  text-transform: uppercase;/*  */
  letter-spacing: .09em;/*  */
  border-radius: 2px;/*  */
  cursor: pointer;/* */
}
.flex-form li {
  padding: 8px;
  font-weight: 300;
  letter-spacing: .09em;
  text-transform: uppercase;
}
  1. Que fait le sélecteur .flex-form > li > label ?
  2. Que fait le sélecteur * ?
  3. Que fait le sélecteur .flex-form > li > label + * ?
  4. Que fait le sélecteur .flex-form li button ?
  5. A quoi correspond la propriété flex-grow ?
  6. A quoi correspond la propriété flex-shrink ?
  7. A quoi correspond la propriété flex-basis ?
  8. A quoi correspond la propriété cursor ?

Exercice 3 - Conception d'une page produits

Bloc de travail

Créer une page HTML en ajoutant le code suivant dans le body :

				 <!-- Partie Header -->
			<nav class="product-filter">			
			  <h1>Jackets</h1>
			  <div class="sort">
				<div class="collection-sort">
				  <label>Filter by:</label>
				  <select>
					<option value="/">All Jackets</option>
				  </select>
				</div>
				<div class="collection-sort">
				  <label>Sort by:</label>
				  <select>
					<option value="/">Featured</option>
				  </select>
				</div>
			  </div>			  
			</nav> 
				 <!-- Partie produits -->
			<section class="products">
			  <div class="product-card">
				<div class="product-image">
				  <img src="img/nature-q-c-70-70-9.jpg">
				</div>
				<div class="product-info">
				  <h5>Winter Jacket</h5>
				  <h6>$99.99</h6>
				</div>
			  <!-- d’autres produits -->

			</section>

Utilisez une nouvelle page css et ajouter le code suivant :

				/* Partie header */
				.product-filter {
				  display: flex;
				}
				.product-filter h1 {
				  flex-grow: 1;/*  */
				}
				.sort {
				  display: flex;/*  */
				}
				.collection-sort {
				  display: flex;/*  */
				  flex-direction: column;/*  */
				}

Cette partie de code permet de gérer le conteneur .product-filter, le titre et les critères de tri (sort en anglais)

  1. Pourquoi <!-- Partie produits --> ne s'affiche pas dans la page HTML ?
  2. Que fait la partie .product-filter h1 ?
  3. Comment se positionne la class sort ?
  4. Dans la class .collection-sort que fait la propriété flex-direction ?

A la suite de votre code css rajoutez celui qui suit.

			/* Partie produits */
				.products {
				  display: flex;
				  flex-wrap: wrap;
				}
				.product-card {
				  padding: 2%;
				  flex-grow: 1;
				  flex-basis: 16%;/* 5 images x 20% =100% + padding 2% =20-4=16%*/
				}
				.product-image img {
				  max-width: 100%; /*Pour nous assurer que les images seront bien adaptées à leur espace*/
				}
				.product-info {
				  margin-top: auto;
				  text-align: center;
				}

L'instruction flex-basis permet de déterminer la taille du bloc produit que l'on souhaite donner, afin de déterminer le nombre d'images que l'on souhaite avoir par ligne.

Si vous souhaitez avoir 5 images par lignes la taille du bloc est de 20% car 5*20% = 100%

Mais le bloc visible, celui de la class product-card est inclus dans le bloc de la class product. Si vous souhaitez avoir un padding de 2%, à droite et à gauche la taille du bloc product-card devient donc de 20-2*2 = 16%

  1. Quelle est la valeur de flex-basis si je veux 10 images par lignes ave un padding à 1% ?
  2. Pourquoi utiliser dans la class product-info la propriété text-align au lieu de justify-content ?
