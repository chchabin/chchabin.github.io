Le code css

		nav ul {
		list-style-type : none; /* Car sinon les puces se placent n'importe où */
		}
		nav ul a {
		text-decoration: none;/* Supprime les soulignements */
		}
		/* Je sélectionne les li du menu pour avoir une présentation horizontal */
		nav ul {
		display: flex;
		}
		/* Je sélectionne les li du menu pour avoir des espaces entre les li */
		nav ul li {
		flex-grow: 1;
		flex-shrink: 1;
		flex-basis: 100%;
		}

Il est possible aprés de changer : les designs, les couleurs etc...
